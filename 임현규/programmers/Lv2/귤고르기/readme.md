# 귤 고르기

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/138476?language=python3)

## 문제 요구 사항

### 문제

- 귤의 크기가 주어진다.
- k 만큼 귤을 뽑는다.
- 뽑은 귤은 최소한의 가짓수를 가져야 한다.

### 제한 사항

- 귤 크기 1 ~ 1000만
- 귤 길이 1 ~ 10만

## 문제 해결 전략

- 빈도수를 구한 뒤 가장 많은 빈도수를 가진 숫자부터 뽑으면 최소 가짓수의 크기를 가진 귤을 박스에 담을 수 있다
