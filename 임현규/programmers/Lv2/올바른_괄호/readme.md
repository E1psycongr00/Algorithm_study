# 올바른 괄호

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12909?language=java)

## 문제 요구 사항

### 문제

- '('로 열리면 ')' 로 닫혀 있어야함.
- 그렇지 않은 경우는 다 옳바르지 않음.

### 제한 사항

- 주어진 문자열의 길이는 10만 이하의 자연수 O(N), O(N log N) 안에 해결해야 함.
- 문자열 s는 '(' 또는 ')'로만 이루어져 있음.

## 문제 해결 전략

- stack 활용 방법
- 괄호 카운트 방법
