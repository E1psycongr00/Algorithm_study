# 주차 요금 계산

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/92341?language=java)

## 문제 요구 사항

### 문제

- 요금표
  - 기본 시간 180분, 요금 5000원
  - 단위 시간 10분, 요금 600원
- 기본 시간 이하면 기본 요금 5000원만 냄
- 기본 시간을 초과하면 기본 요금 + [초과 분 / 10] * 600 만큼의 돈을 냄

### 제한 사항

- fees 길이 4, records 길이 1 ~ 1000

## 문제 해결 전략

- 실 서비스에 있을 법한 단순 구현 문제
